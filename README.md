# ShelfStock — frontend (deployment mirror)

This repository is the **Vercel deployment source** for the ShelfStock storefront.

📦 **The canonical repository is [jasrulete/Shelfstock](https://github.com/jasrulete/Shelfstock)** —
the full monorepo with documentation, unit + end-to-end tests, Docker Compose
setup, and CI running all of it on every push.

🔗 **Live demo:** [shelfstock-jer2x.vercel.app](https://shelfstock-jer2x.vercel.app/)

Next.js 14 (App Router) + TypeScript + Tailwind. Highlights: debounced search
with AbortController, server-side pagination, multi-currency display with
cached exchange rates, and a localStorage cart synced across tabs.
